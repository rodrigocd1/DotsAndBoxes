## Regras Éticas de Monetização

### Permitido

- Anúncios recompensados opcionais
- Assinatura com benefícios claros
- Compra única para remover anúncios forçados
- Preços transparentes

### Proibido

| Prática | Motivo |
|---|---|
| Frase "2 meses grátis" | Marketing enganoso |
| Preço falso riscado | Dark pattern |
| Loot box paga | Sensação de cassino |
| Roleta paga | Sensação de cassino |
| Recompensa aleatória comprada | Sensação de cassino |
| Criar sensação de cassino | Antiético |
| Esconder preço real | Falta de transparência |

---

## Anúncios

- Anúncios recompensados são sempre **opcionais**
- "Remover anúncios" remove banners e anúncios forçados
- Anúncios recompensados **continuam disponíveis** após "Remover anúncios" (são opcionais por natureza)
- Banners NÃO aparecem em: menu principal, feedback, tutorial, gameplay, ranked, X1
- Banners aparecem em: loja, mapa, perfil (pequeno/fixo)

---

## Regras de Exibição de Banner

- Nunca esconder banner
- Nunca cortar banner
- Nunca deixar "frestinha"
- Nunca posicionar para impressão artificial
